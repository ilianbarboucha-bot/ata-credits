# ATA Credits Site

Minimal static public site for ATA Credits.

Files:

- `index.html`
- `privacy.html`
- `terms.html`
- `styles.css`

This folder is intentionally build-free so it can be:

- served locally with any static server;
- deployed to GitHub Pages or another static host;
- used immediately as the public-facing product URL for provider onboarding.
