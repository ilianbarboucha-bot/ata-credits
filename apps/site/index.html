How it works

Built for AI waits, not prompt resale.

1. Estimate first

ATA Credits estimates each request before execution and chooses the route before the request starts.

2. Recharge safely

Sponsor cards appear during wait time. Credits enter pending first, then move to available only after delayed validation.

3. Keep prompts private

Ad providers receive only placement-safe metadata such as tool, route, category, approximate country, and anonymous session identifiers.
